# CineQL

A web app that fetches movie details from an [movie API](https://yts.am/api)

## Built with
- JavaScript
- Node.js
- GraphQL ([GraphQL-Yoga](https://github.com/prisma/graphql-yoga))

## Things that you can fetch from the API
- id
- title
- rating
- summary 
- language
- cover image
- genres
